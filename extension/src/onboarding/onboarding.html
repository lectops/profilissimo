I

Install the helper app

A small local binary so Profilissimo can talk to Chrome. Open Terminal and run:

curl -fsSL https://raw.githubusercontent.com/lectops/profilissimo/main/installer/install.sh | bash

One install serves every Chrome profile on this Mac. Or download manually →

II

Restart Chrome

Quit and reopen Chrome (⌘Q) — Chrome only picks up the helper app at startup.

III

Verify connection

A quick check that everything’s wired up.

p

You’re ready.

Profilissimo is connected. Open the popup or use the keyboard shortcut anytime.

P.S. Right-click any page and pick “Always open this site in…” to pin a hostname to a profile.
Step IV — and last

Install in your other Chrome profiles.

Chrome doesn’t let an extension install itself across profiles. Profilissimo will open the Web Store in each — just click Add to Chrome.
