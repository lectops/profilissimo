Welcome to Profilissimo

Open tabs in any Chrome profile with one click.

✓ Helper app connected
1

Install the helper app

Profilissimo needs a small helper app to communicate with Chrome profiles. Open Terminal and run this command:

curl -fsSL https://raw.githubusercontent.com/lectops/profilissimo/main/installer/install.sh | bash

You only need to do this once — the helper app works across all your Chrome profiles. Or download manually.

2

Restart Chrome

Quit Chrome completely (Cmd+Q) and reopen it. Chrome only detects new helper apps on startup.

3

Verify connection

Click the button below to check that everything is working.

✓

You're all set!

Profilissimo is connected and ready to use.

New: right-click any page and pick "Always open this site in…" to auto-redirect that hostname to a chosen profile. Manage rules in Settings.

One last step — add Profilissimo to your other profiles

Chrome won't let an extension install itself in other profiles — but Profilissimo can open the Web Store in each of your other profiles in one go. You'll just click Add to Chrome in each window.
