I

Install the helper app

A tiny local program that lets Profilissimo open your other Chrome profiles. Open Terminal and paste this one line:

Terminal
$ curl -fsSL profilissimo.app/install | bash
Safe to run. It installs one small helper into your home folder and nothing else — no admin password, no system changes. Prefer to download it yourself? →
II

Restart Chrome

Quit and reopen Chrome ⌘Q — it only notices the helper at startup.

III

Verify the connection

A quick check that everything's wired up.
p
All set

You're ready.

Profilissimo is connected. Open the popup or hit the shortcut anytime.

P.S.
Right-click any page and choose “Pin this site to…” to make a site always open in one profile.

Add it to your other Chrome profiles.

Chrome runs each extension per profile, so it can't install itself across them. We'll open the Web Store in each — just click Add to Chrome.
